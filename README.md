# KIZZ SHOP GAME - Premium Game Top-Up Platform

KIZZ SHOP GAME is a high-performance web application designed for game top-up services (such as Mobile Legends Diamonds, PUBG UC, and more). This project focuses on providing a seamless user experience with a modern interface and a reliable payment selection system.

## 📌 About This Project
This project was developed as a practical implementation of modern front-end technologies. It serves as a digital storefront for "KIZZ SHOP GAME," allowing users to select their favorite games, choose denominations, and pick their preferred payment method through an interactive UI.

## ✨ Core Features
*   **Dynamic Payment Integration**: A robust payment selection system featuring QRIS, DANA, OVO, and GoPay.
*   **Automated Fee Calculation**: Displays real-time admin fees for each payment method.
*   **Responsive Game Catalog**: A grid-based display of popular games with high-quality banners.
*   **Interactive UI Components**: Built with reusable React components for a fast and smooth user experience.
*   **Local Storage Integration**: Implements local storage for managing task lists and user data persistence.

## 💻 Tech Stack & Libraries
*   **Language**: JavaScript (ES6+)
*   **Framework**: React.js (Vite)
*   **Styling**: CSS3 (Custom Flexbox & Grid System)
*   **Navigation**: React Router DOM (for seamless page transitions)
*   **Icons**: Lucide React / React Icons
